FAQ:

What is read.eurostat?
  It is a function in R to facilitate the download of data from eurostat


How to use read.eurostat
  When browsing for eurostat data you find the dataset code  
  e.g. [lfsa_ewhuis] in the upper right corner.
  call read.eurostat("lfsa_ewhuis","en")  to downloaded a labeled dataset in English.
  Use "fr" or "de" for French or German respectively.
